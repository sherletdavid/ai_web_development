Tech-Innovate 2025 Event Portal
  This repository hosts the front-end code for the Tech-Innovate 2025 event registration and promotional portal. The portal provides information about the event, including     promotional media, and a registration form for attendees. 
  
Project Structure
  The project consists of a single HTML file (index.html - which is the provided code snippet), utilizing basic HTML5, CSS, and form elements to create an interactive          registration experience. 
  
Features
  Event Promotion: Includes an image, a video placeholder, and an audio placeholder to promote event highlights.
  Registration Form: A comprehensive form to capture attendee details.
  Full Name (required)
  Email Address (required)
  Preferred Session Date (required)
  Attendee Type (Student/Professional radio buttons)
  Workshop Selection (dropdown menu)
  Motivation Textarea
  ID Proof Upload (accepts PDF or images) 

  
Usage
  To run this portal locally:
  Clone this repository to your local machine.
  Open the index.html file in your preferred web browser. 
  Note on Media Files: The provided HTML references local file paths for the image, video, and audio elements (Lab10.jpg, C:\Users\STUDENTS\EXp1\vid1.mov, audio.mp3,           via.placeholder.com). To make the media elements functional, you must: 
  Replace the placeholder src attributes with actual paths to your media files.
  Ensure these media files are included in your project directory (or hosted online and linked accordingly). 
  
Technologies Used
  HTML5: For the structure and form elements.
  CSS: For styling and layout (embedded within the <style> tags)
